I personally found the use of a "CPU" inside Minecraft rather interesting<br/>
Most of the challenges in this folder are related to using the mc86 Minecraft CPU<br/>

Below is the description from the organisers:<br/>
Our mc86 problems introduce a new vanilla Minecraft CPU (java edition).<br/>
It helps to have Java Edition Minecraft but if you don't they can still be solved the old fashioned way.<br/>
